# Los Angeles Crime Data Analysis ![Status](https://img.shields.io/badge/status-complete-brightgreen)

This project was completed for the course **Intro to Data Analysis**.
The project analyzes Los Angeles crime data from 2020–2023. It includes exploratory data analysis, victim demographic analysis, time-based crime trends, hypothesis testing, and a KNN predictive model.


## Main Methods

- Data cleaning and preprocessing
- Exploratory data analysis
- Data visualization
- Hypothesis testing
- KNN classification
- Cross-validation

## Tools

- Python
- pandas
- NumPy
- matplotlib
- seaborn
- scikit-learn

## Dataset

The project uses Los Angeles crime data from 2020–2023.  
The raw dataset is not included because of file size and source limitations.


## Files

- `LAPD_Crime_Rates.ipynb` - main notebook
- `LAPD_Crime_Rates.pdf` - final project report
- `requirements.txt` - required Python packages

## How to Run

Open `LAPD_Crime_Rates.ipyn` and install the packages listed in `requirements.txt` using:
```bash
pip install -r requirements.txt
```
then run each notebook cell from top to bottom.
